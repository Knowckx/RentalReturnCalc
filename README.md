## Minimal React template

A minimal React.js template that includes the following technology stack:

- React18
- TypeScript
- Vite
- pnpm
- tailwindcss4.0
- shadcn-ui
- PWA
- https

First Page is a bottom-navigation bar.

pnpm run build

pnpm preview --host
